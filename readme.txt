Программа позволяет генерировать произвольное облако слов на основе веб-страницы с текстом формата *.txt или страницы Википедии, название которой вводится при выполнении программы.
При этом текст очищается от лишних символов с помощью регулярных выражений (библиотека re), очищается от стоп-слов и лемматизируется с помощью библиотеки nltk.
